About

This is a simple parallax website created as a project demo. It uses background-attachment fixed technique to give a parallax scrolling effect. Great for portfolio or landing pages.

Help

If you face issues viewing the parallax sections on mobile devices, try viewing on a desktop. Images may not scroll as expected due to device compatibility with background-attachment property.

Info

Developed using HTML and CSS only. No JavaScript was required. You can customize the background images, text, and sections as needed for your own project.
